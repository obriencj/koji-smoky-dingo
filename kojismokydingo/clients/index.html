Koji Smoky Dingo - Client Utils

Some simple functions for working with the local client configuration

license:

rebuild_client_config(session, goptions)[source]¶

Reconstructs a koji client configuration based on the fields of a session and a session's goptions. Returns a tuple containing the active profile's name, and the configuration as a dict.

Parameters:

  • session (ClientSession) -- a koji client session

  • goptions (GOptions) -- the koji CLI's options

Since:

1.0

Return type:

Tuple[str, Dict[str, Any]]
